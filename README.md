## About

This page provides a searchable view of the top 20 AI research papers each week from:
- **arXiv**: cs.AI, cs.LG, cs.CL, cs.CV, cs.NE
- **Nature**: Machine Intelligence, AI articles
- **Science**: AI/ML research
- **Conference proceedings**: PMLR, OpenReview

Papers are automatically categorized into:
- Model
- Evaluation
- Hardware/Infra
- Policy
- Industry
- Incident
- Project

## Data Source
